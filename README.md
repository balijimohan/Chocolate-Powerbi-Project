#Chocolate Sales Power BI Project Report
##Project Objective
To analyze chocolate sales data and create interactive dashboards that help understand business performance and
improve decision-making
##Tools & Technologies Used
• Power BI
• Excel / CSV Dataset
• DAX Measures
• Data Cleaning & Transformation
##Dataset Details
The dataset contains information related to chocolate product sales, revenue, profit, quantity sold, customer regions,
and monthly trends.
